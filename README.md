# processTCP
Get TCP Connected Processes of Local IP Address

You can sort output by 
   1 = OwningProcess [default]
   2 = LocalPort
   3 = RemoteAddress
   4 = RemotePort
   5 = State
   6 = ProcessName
   7 = ServiceName
   8 = LocalAddress

You can sort order by
   1 = Ascending [default]
   2 = Descending
   
You can filter IP address by 
   1 = Current IP address [default]
   2 = 127.0.0.1
   3 = 0.0.0.0
   0 = ALL IP [no filter]
